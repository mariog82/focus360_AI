
# Incident Response Plan sintetico

1. Identificazione evento.
2. Classificazione severità.
3. Contenimento tecnico.
4. Analisi forense e preservazione log.
5. Comunicazione a DPO/dirigente/fornitori.
6. Valutazione data breach GDPR entro 72 ore se applicabile.
7. Valutazione obblighi NIS2/D.Lgs. 138/2024 se applicabili.
8. Remediation e lesson learned.
