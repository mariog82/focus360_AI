
# DevOps / DevSecOps pipeline Focus360 AI Alpha Microservices

1. **Plan**: backlog per microservizio, threat modeling, privacy-by-design, DPIA preliminare.
2. **Code**: feature branch, code review, secret scanning, SAST.
3. **Build**: build container per servizio, SBOM, dependency pinning.
4. **Test**: unit, integration, system, acceptance, security baseline.
5. **Release**: versioning semantico, changelog, migrazioni database controllate.
6. **Deploy**: ambienti dev/staging/prod, blue-green o rolling deployment.
7. **Operate**: logging, audit, backup, incident response, SLA.
8. **Monitor**: metriche tecniche, sicurezza, accessi, disponibilità, error budget.
